## Problem Statement
Detect and recognize human faces in real-time using a hybrid approach combining Haar Cascade (for detection) and CNN (for recognition).

---

## Approach
- **Haar Cascade** — Fast frontal face detection
- **CNN Model** — Deep learning-based face recognition
- **OpenCV** — Real-time video processing

---

## Project Structure
| File | Description |
|------|-------------|
| `collect_data.py` | Captures face images for training |
| `consolidated_data.py` | Prepares and consolidates dataset |
| `face_detection.py` | Main detection script |
| `recognize.py` | Real-time face recognition |
| `final_model.h5` | Trained CNN model |
| `haarcascade_frontalface_default.xml` | Haar Cascade config |
